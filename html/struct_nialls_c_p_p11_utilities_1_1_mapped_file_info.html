Detailed Description

Information about mapped files in the process.

This is not a fast call, on any system. On Linux and FreeBSD this call returns a perfect snapshot - on Windows and Mac OS X, there is a slight possibility that data returned is incomplete or contains spurious data as the set of mapped files may change mid-traversal.

To use this you must compile MappedFileInfo.cpp and ErrorHandling.cpp.
